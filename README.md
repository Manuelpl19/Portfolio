#  - Gestión de Tareas

## Descripción General

Este es un Gestor de Tareas moderno y sencillo implementado como una **SPA**. El sistema está diseñado con una arquitectura desacoplada, utilizando **Laravel** exclusivamente para la lógica de la API y la persistencia de datos, y **React** para ofrecer una interfaz de usuario dinámica y fluida.

### Tecnologias

Este proyecto se basa en un stack moderno y optimizado para rendimiento y escalabilidad:

| **Backend** | Servidor de Datos y Lógica de Negocio | **Laravel** (PHP) |
| **Frontend** | Interfaz de Usuario y Lógica de Presentación | **React** (JavaScript) |
| **Autenticación** | Gestión de Sesiones y Tokens | **Laravel Sanctum** |
| **Estilos** | Diseño responsivo y utilitario | **Tailwind** |
| **Comunicación** | Peticiones HTTP del Frontend al Backend | **Fetch API** |
